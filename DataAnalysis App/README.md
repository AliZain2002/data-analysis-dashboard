# Data Analysis Dashboard

A multi-page Dash application for data preprocessing and visualization.

## Features
*   **Upload Data**: Drag and drop CSV files.
*   **Preprocessing**: Handle missing values, change data types, and drop columns.
*   **Univariate Analysis**: Histograms with bin sliders and box plots.
*   **Bivariate Analysis**: Scatter and Line plots.
*   **Multivariate Analysis**: Correlation heatmaps.
*   **Download**: Export cleaned data.

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `python index.py`
